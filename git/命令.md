#### autocrlf 配置

```javascript
// git 提交时将 crlf 转成 lf, 检出时将 lf 转成 crlf(windows)
git config --global core.autocrlf true
// git 提交/检出时都不做转换
git config --global core.autocrlf true
// git 提交时将 crlf 转成 lf, 检出时不转换(mac)
git config --global core.autocrlf input
```
